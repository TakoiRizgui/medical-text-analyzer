# 🧠 Medical Text Analyzer (NLP Project)

## 📌 Project Overview
This project aims to analyze medical text data using Natural Language Processing (NLP) techniques.

The goal is to classify medical reports into:
- Normal cases
- Abnormal cases

## ⚙️ Methodology
- Text preprocessing
- TF-IDF vectorization
- Logistic Regression model

## 📊 Results
- The model successfully classifies medical texts
- Key terms influencing predictions are extracted

### 🔍 Feature Importance
The model identifies important words associated with:
- Abnormal cases (e.g. fever, glucose, infection)
- Normal cases (e.g. normal, routine)

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
